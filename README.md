# ASCII Studio

ASCII Studio is a Python-based tool that blends retro terminal aesthetics with modern design. It converts images into character art, generates stylized text banners, and exports graphics using standard character sets.

## Features
* **Image to Art:** Convert photos into detailed symbol-based graphics.
* **Text Banners:** Create retro headers using custom monospace fonts.
* **Easy Export:** Copy directly to your clipboard or save as `.txt` files.

## Prerequisites
* Python 3.8 or higher
* Pillow library (for image processing)

## Installation & Setup on Linux

Follow these steps to set up and run ASCII Studio on any Linux distribution (Ubuntu, Debian, Fedora, etc.):

1. **Update packages and install Python essentials:**
   ```bash
   sudo apt update && sudo apt install python3 python3-venv python3-pip -y
   ```

2. **Clone the repository and navigate to the project folder:**
   ```bash
   git clone git clone https://github.com/XP-2001/...
   cd ascii-studio
   ```

3. **Create and activate a virtual environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

4. **Install the required dependencies:**
   ```bash
   pip install Pillow
   ```

5. **Launch the application:**
   ```bash
   python3 main.py
   ```
   *(Note: Replace `main.py` with the actual name of your script).*
