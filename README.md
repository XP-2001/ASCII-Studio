# ASCII Studio

ASCII Studio is a Python-based tool that blends retro terminal aesthetics with modern design. It converts images into character art, generates stylized text banners, and exports graphics using standard character sets.

## Features
* **Image to Art:** Convert photos into detailed symbol-based graphics.
* **Text Banners:** Create retro headers using custom monospace fonts.
* **Easy Export:** Copy directly to your clipboard or save as `.txt` files.
